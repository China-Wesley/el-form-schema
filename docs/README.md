## 安装

```shell
yarn add https://github.com/China-Wesley/el-form-schema.git
```

### 引入 el-form-schema

```javascript
import ElFormSchema from 'el-form-schema'
import 'el-form-schema/dist/el-form-schema.css'

Vue.use(ElFormSchema)
```
